# Southeast Asian Haze Crisis Dashboard

A comprehensive, professional dashboard analyzing transboundary haze pollution in Malaysia and Southeast Asia.

## 🌐 Live Demo

Visit the live dashboard: [https://yourusername.github.io/haze-dashboard-malaysia](https://yourusername.github.io/haze-dashboard-malaysia)

## 📊 Features

### Interactive Components
- **Historical Timeline**: Major haze episodes from 1972-2025
- **Interactive Maps**: Geographic visualization of haze severity across Malaysian states
- **Economic Analysis**: State-level economic impact assessment
- **Health Impact Data**: Comprehensive health statistics and healthcare utilization
- **Policy Tracking**: Government initiatives and budget allocations
- **NGO Activities**: Civil society efforts and environmental advocacy
- **Public Opinion**: Survey data and policy reaction analysis

### Technical Features
- Professional design with Times New Roman typography
- Responsive layout for desktop and mobile devices
- Interactive Leaflet.js maps with detailed state information
- Chart.js visualizations for historical data trends
- Comprehensive source citations and academic references
- No external dependencies beyond CDN resources

## 🚀 Quick Start

### GitHub Pages Deployment

1. **Fork this repository**
2. **Enable GitHub Pages**:
   - Go to Settings → Pages
   - Source: Deploy from a branch
   - Branch: main, Folder: / (root)
3. **Access your dashboard**: `https://yourusername.github.io/haze-dashboard-malaysia`

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/haze-dashboard-malaysia.git
   cd haze-dashboard-malaysia
   ```

2. **Open locally**:
   - Simply open `index.html` in your web browser, or
   - Use a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # PHP
   php -S localhost:8000
   ```

3. **Access**: `http://localhost:8000`
